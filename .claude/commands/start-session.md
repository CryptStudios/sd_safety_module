Start a content-authoring session for the SD Safety Module.

Follow the **`/start-session`** procedure defined in [CLAUDE.md](../../CLAUDE.md):

1. Read `docs/STATUS.md` to find the next category and any decision/blocker.
2. Read `docs/CONTENT_PLAN.md` for that category's primary source and scope.
3. Make sure the dev server is running at http://localhost:3000 (start it if not).
4. Define or confirm the category's ~11 real toolbox-talk topic titles.
5. State the session goal: category, topic count, primary source — then wait for
   my go-ahead before authoring.

Also review the **Article-writing methodology** in CLAUDE.md before writing any content.
